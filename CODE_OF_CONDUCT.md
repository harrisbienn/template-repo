# Code Of Conduct

This repository should be a professional, inclusive, and respectful place to collaborate.

## Expectations

- Assume good intent, but clarify quickly when expectations are ambiguous.
- Review work by focusing on correctness, maintainability, security, and user impact.
- Keep client, project, and personal information confidential.
- Raise risks early, especially around data handling, security, and production operations.

## Reporting

Report conduct concerns through the organization's established people or project leadership channels.
